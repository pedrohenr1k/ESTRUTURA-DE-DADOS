# Estruturas de Dados

Este repositório contém implementações de:

- **Listas Encadeadas**
- **Listas Ordenadas**
- **Árvores Binárias**

## Objetivo

O objetivo é estudar e implementar listas encadeadas, listas ordenadas e árvores binárias
